# Financial Data Extraction & Visualization Using Python

## Overview
This project extracts daily stock price data from web APIs and visualizes trends, moving averages, and comparative performance across companies.

## Objectives
- Fetch real-time stock data  
- Perform EDA on financial time series  
- Compute moving averages and performance metrics  
- Build clear financial visualisations  

## Workflow Summary
1. **Data Extraction**
   - Import stock data (using requests or pandas-datareader)

2. **Data Cleaning**
   - Parse dates  
   - Handle missing values  

3. **Visual Analysis**
   - Line charts  
   - Moving averages  
   - Adjusted close comparisons  

4. **Interpretation**
   - Trend analysis  
   - Volatility observations  

## Results
- Successfully extracted and visualised multi-stock trends.
- Moving averages highlighted market momentum patterns.

## Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Pandas DataReader
